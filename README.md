# RSS Feed Roll

RSS Feed of Engineering Blogs Inspired by Feedly.

### Getting Started

 ```
 git clone https://github.com/mystica2000/feed-roll.git
 ```

 #### Backend

  1. Navigate to directory
   ```
    cd backend
   ```

  2. Install Dependencies
   ```
    go mod tidy
   ```

  3. Start Server
   ```
   go run server.go
   ```

### Built with
 - Go
 - SolidJS
